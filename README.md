# Quant Research Book

This repository is dedicated to quantitative finance research. It contains exploratory studies, backtesting experiments, strategy prototypes, and data analysis notebooks covering various quantitative trading ideas. The primary goal is to share knowledge, foster learning, and document research findings in a reproducible way.
